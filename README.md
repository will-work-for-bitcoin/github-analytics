# github-analytics

GitHub repository analytics CLI - stars, forks, contributors, language detection, and JSON export.

## Installation

```bash
git clone https://github.com/will-work-for-bitcoin/github-analytics.git
cd github-analytics
```

## Usage

```bash
python github_analytics.py
python github_analytics.py --help
```

## Features

- Pure Python 3.7+, zero dependencies
- JSON/CSV export support
- Beautiful terminal output
- Easy to extend and customize
